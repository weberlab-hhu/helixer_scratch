\section{Introduction}
\begin{itemize}
	\item Calling genes correctly is very important
	\item but also very difficult in eukaryotes
	\item All current state of the art gene callers work based on HMMs
	\item The structure of the HMMs is finely tuned and engineered
	\item Actual learning is very limited, and with that generalization across different genomes 
	\item (Also have problems with genes that contain long introns, common in mammals)
\end{itemize}

\section{Approach}
We seek to show a path for improving on current HMM-based methods both in performance and in the ability to generalize. To accomplish this, we use deep neural networks, which have been shown to have the potential to be superior in flexibility and accuracy to traditional machine learning methods when detecting complex patterns in variety of fields.
